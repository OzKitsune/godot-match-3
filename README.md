Реализация игры **Match-3** на языке **C#** с использованием .NET-версии **Godot Engine 4.6.1**.

## Сборка и запуск

1. Клонируйте репозиторий:
   
   ```git clone https://github.com/OzKitsune/godot-match-3.git```

2. Откройте проект в Godot Engine 4.6.1 (импортируйте его как существующий проект).

3. Нажмите `F5` для запуска игры.

Или загрузить сборку из раздела Releases.

### Основные скрипты (`source/`)

| Класс             | Назначение                                                                                                             |
| ----------------- | ---------------------------------------------------------------------------------------------------------------------- |
| `Game.cs`         | Управляет состоянием игры, таймером, счётом и переходами.                                                              |
| `Board.cs`        | Ядро игровой логики: сетка элементов, поиск совпадений, обмен, анимации, создание и активация бонусов.                 |
| `AGameElement.cs` | Абстрактный базовый класс для всех игровых элементов (обычных и бонусных). Обрабатывает клики, выделение, уничтожение. |
| `BonusInfo.cs`    | Хранит информацию о будущем бонусе и создаёт его. Использует словарь фабрик для расширяемости.                         |
| `MatchLine.cs`    | Хранит информацию о найденной линии с типом, цветом и координатами.                                                    |
| `Destroyer.cs`    | Разрушитель, уничтожающий элементы на своём пути.                                                                      |
